Savoir-faire

  • Manipuler le DOM : récupérer des éléments avec getElementById, changer leur contenu avec innerHTML, gérer les événements comme onclick ou l’input utilisateur.

  • Faire des calculs et afficher les résultats directement dans la page, convertir les données quand il faut.

  • Récupérer et afficher des données depuis des API (WorldTimeAPI, API anime), traiter les infos pour que ça s’affiche correctement.

  • Créer des interfaces modulaires : mes bulles réutilisables, mes boutons animés et mes logos animés.

  • Rendre les sites responsive avec des media queries, pour que ça marche sur grand ou petit écran.

  • Utiliser les variables CSS pour avoir un design cohérent et gérer les animations hover et clic.

  • Structurer un projet front-end propre : fichiers bien séparés (HTML, CSS, JS), logique claire, code réutilisable quand c’est possible.

  • Mettre à jour du contenu dynamique sans toucher au HTML statique, par exemple pour des listes ou des résultats automatiques.
